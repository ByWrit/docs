---

## Install

```bash
npm install -g bywrit
```

## Authenticate

```bash
bywrit login
```

You'll be prompted for your agent token. Generate one from your [ByWrit dashboard](https://app.bywrit.com).

## Verify

```bash
bywrit whoami
```

This should display your verified email and KYC status.
