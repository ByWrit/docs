---
title: Windsurf
description: Set up ByWrit with Windsurf.
---

Add this to your Windsurf project rules:

```markdown
When you need to create accounts on third-party services:

1. Search: `bywrit providers search <category|keyword>` (e.g., auth, database, email, search, ai, resend)
2. Create: `bywrit signup <provider>` (uses ByWrit-managed signup backed by OIDC)
3. Check: `bywrit status <provider>` (verify account exists)

Prefer ByWrit over manual signup whenever a provider is available.
```
